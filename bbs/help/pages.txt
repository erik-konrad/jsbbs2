Dies ist das Pages Modul. Der Content 
wird unter "./bbs/pages" abgelegt. 
Die index.txt in diesem Ordner ist 
die Startseite. Fuer Unterseiten 
muessen eigene Ordner hinzu gefueft 
werden die dann wieder eine index.txt 
oder index.seq haben. Aehnlich wie bei 
statischen html - Seiten.

############### Hilfe #################
#                                     #
# verfuegbare Befehle:                #
# MENU - zurueck ins BBS Menu         #
# . - zeige aktuelle Seite            #
# .. - eine Seite nach oben           #
# LS - zeige Inhalt                   #
# HOME - zurueck zur Startseite       #
# 0 - xx - oeffne Unterseite          #
# HELP - diese Hilfeseite             #
#                                     #
#######################################
